## **1. Hlavn√≠ t√©mata**

### **1.1 Popisn√° statistika**

#### **Statistiky polohy**
- **Pr≈Ømƒõr (`mean`)**: Souƒçet v≈°ech hodnot vydƒõlen√Ω jejich poƒçtem. Pou≈æ√≠v√° se pro mƒõ≈ôen√≠ st≈ôedn√≠ hodnoty dat.
  - **Nev√Ωhoda:** Citliv√Ω na odlehl√© hodnoty.
  - **P≈ô√≠klad:**
  ```r
  mean(c(1, 2, 3, 4, 100)) # V√Ωsledek je silnƒõ ovlivnƒõn hodnotou 100
  ```

- **Medi√°n (`median`)**: Prost≈ôedn√≠ hodnota dat se≈ôazen√Ωch podle velikosti. Je robustn√≠ v≈Øƒçi odlehl√Ωm hodnot√°m.
  - **P≈ô√≠klad:**
  ```r
  median(c(1, 2, 3, 4, 100)) # Medi√°n nen√≠ ovlivnƒõn hodnotou 100
  ```

- **Modus (`Mode`)**: Nejƒçastƒõji se vyskytuj√≠c√≠ hodnota v datov√© sadƒõ.
  - **Unimod√°ln√≠ rozdƒõlen√≠:** M√° jeden v√Ωrazn√Ω vrchol.
    ```r
    Mode(c(1, 2, 2, 3, 4)) # V√Ωsledek: 2
    ```
  - **Bimod√°ln√≠ rozdƒõlen√≠:** M√° dva v√Ωrazn√© vrcholy.
    ```r
    Mode(c(1, 2, 2, 4, 4, 5)) # V√Ωsledek: 2, 4
    ```
  - **Multimod√°ln√≠ (polymod√°ln√≠) rozdƒõlen√≠:** M√° v√≠ce ne≈æ dva vrcholy.
    ```r
    Mode(c(1, 2, 2, 4, 4, 5, 5)) # V√Ωsledek: 2, 4, 5
    ```
  - **Amod√°ln√≠ rozdƒõlen√≠:** Nem√° ≈æ√°dn√Ω v√Ωrazn√Ω vrchol (rovnomƒõrn√© rozdƒõlen√≠).
    ```r
    Mode(c(1, 2, 3, 4, 5)) # ≈Ω√°dn√° hodnota nem√° vy≈°≈°√≠ frekvenci
    ```

- **Useknut√Ω pr≈Ømƒõr (`mean(trim = 0.1)`)**: Pr≈Ømƒõr vypoƒçten√Ω po odstranƒõn√≠ krajn√≠ch hodnot na obou stran√°ch. Parametr `trim` ud√°v√° procento odstranƒõn√Ωch hodnot.
  - **P≈ô√≠klad:**
  ```r
  mean(c(1, 2, 3, 4, 100), trim = 0.2) # Odstran√≠ 20 % krajn√≠ch hodnot
  ```

#### **Statistiky variability**
- **Smƒõrodatn√° odchylka (`sd`)**: Mƒõ≈ô√≠ pr≈Ømƒõrnou vzd√°lenost hodnot od pr≈Ømƒõru.
  - **P≈ô√≠klad:**
  ```r
  sd(c(1, 2, 3, 4, 5))
  ```

- **Rozptyl (`var`)**: Druh√° mocnina smƒõrodatn√© odchylky.
  - **P≈ô√≠klad:**
  ```r
  var(c(1, 2, 3, 4, 5))
  ```

- **Mezikvartilov√© rozpƒõt√≠ (`IQR`)**: Rozd√≠l mezi t≈ôet√≠m (Q3) a prvn√≠m (Q1) kvartilem. Odoln√© v≈Øƒçi odlehl√Ωm hodnot√°m.
  - **P≈ô√≠klad:**
  ```r
  IQR(c(1, 2, 3, 4, 5))
  ```

- **MAD (`mad`)**: Medi√°n absolutn√≠ch odchylek od medi√°nu. Robustn√≠ statistika variability.
  - **P≈ô√≠klad:**
  ```r
  mad(c(1, 2, 3, 4, 5))
  ```

#### **Statistiky tvaru rozdƒõlen√≠**
- **≈†ikmost (`Skew`)**: Mƒõ≈ô√≠ asymetrii dat.
  - Symetrick√°: ≈†ikmost = 0
  - Pravostrann√°: ≈†ikmost > 0
  - Levostrann√°: ≈†ikmost < 0
  - **P≈ô√≠klad:**
  ```r
  Skew(c(1, 2, 3, 4, 10))
  ```

- **≈†piƒçatost (`Kurt`)**: Mƒõ≈ô√≠, jak moc jsou data ≈°piƒçat√° ve srovn√°n√≠ s norm√°ln√≠m rozdƒõlen√≠m.
  - **P≈ô√≠klad:**
  ```r
  Kurt(c(1, 2, 3, 4, 5))
  ```

### **1.2 Frekvenƒçn√≠ rozdƒõlen√≠ a kumulativn√≠ ƒçetnosti**
- **Absolutn√≠ ƒçetnost (`table`)**: Poƒçet v√Ωskyt≈Ø hodnot.
- **Relativn√≠ ƒçetnost (`prop.table`)**: Pod√≠l jednotliv√Ωch hodnot.
- **Kumulativn√≠ ƒçetnost (`cumsum`)**: Postupn√Ω souƒçet ƒçetnost√≠.
- **P≈ô√≠klad:**
```r
x <- c('A', 'B', 'A', 'C', 'B', 'A')

data.frame(cbind("n(i)"=tab,"N(i)"=kac,"f(i)"=prop.table(tab),"F(i)"=krc))
```

# Rozdƒõlen√≠ pravdƒõpodobnosti

## Pravdƒõpodobnostn√≠ funkce
- Vyjad≈ôuje pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° nabyde pr√°vƒõ dan√© hodnoty x (=)
- P≈ô√≠klad: Pokud hod√≠me kostkou, jak√° bude pravdƒõpodobnost, ≈æe padne pr√°vƒõ ƒç√≠slo 4 (x = 4)

## Distribuƒçn√≠ funkce
- Vyjad≈ôuje pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° nabyde men≈°√≠ nebo rovno dan√© hodnoty x (<=)
- P≈ô√≠klad: Pokud hod√≠me kostkou, jak√° bude pravdƒõpodobnost, ≈æe padne ƒç√≠slo men≈°√≠ nebo rovno 4 (x <= 4)

## Pravdƒõpodobnost√≠ vs Distribuƒçn√≠ funkce
- distribuƒçn√≠ funkce nen√≠ nic jin√©ho ne≈æ suma pravdƒõpodobnostn√≠ch funkc√≠
- P(x ‚â§ 3) = P(x = 0) + P(x = 1) + P(x = 2) + P(x = 3)
- ƒçili, jak√°koliv √∫loha, kterou lze vy≈ôe≈°it pomoc√≠ distribuƒçn√≠ funkce lze z√°rove≈à p≈ôev√©st na ≈ôe≈°en√≠ pomoc√≠ pravdƒõpodobnostn√≠ch funkc√≠
- distribuƒçn√≠ funkce P(x ‚â§ 3) vlastnƒõ vypoƒç√≠t√°v√° pravdƒõpodobnost, ≈æe ƒçty≈ôka padne 0, 1, 2 nebo 3 kr√°t
- v√Ωstup distribuƒçn√≠ funkce se t√©≈æ nƒõkdy naz√Ωv√° kumulativn√≠ pravdƒõpodobnost (co≈æ nyn√≠ d√°v√° smysl - kumuluje v√Ωsledky pravdƒõpodobnostn√≠ch funkc√≠, kter√© postupnƒõ sƒç√≠t√°)

```r
# Pravdƒõpodobnost, ≈æe kdy≈æ hod√≠m kostkou 10kr√°t, tak padne nap≈ô. ƒçty≈ôka maxim√°lnƒõ 3kr√°t (binomick√© rozdƒõlen√≠).

k = 3
n = 10
p = 1/6

# Vypoƒç√≠t√°no distribuƒçn√≠ funkc√≠
# V√Ωsledek: 0.9302722
pbinom(q = k, size = n, prob = p)

# Vypoƒç√≠t√°no sumou/souƒçtem pravdƒõpodobnostn√≠ch funkc√≠
# V√Ωsledek: 0.9302722
dbinom(x = 0, size = n, prob = p) + dbinom(x = 1, size = n, prob = p) + dbinom(x = 2, size = n, prob = p) + dbinom(x = 3, size = n, prob = p)

# Zkr√°cen√Ω z√°pis p≈ô√≠padu v√Ω≈°e
# V√Ωsledek: 0.9302722
sum(dbinom(x = 0:k, size = n, prob = p))
```

# Diskr√©tn√≠ rozdƒõlen√≠
- maj√≠ integer hodnoty (diskr√©tn√≠)
- P≈ô√≠klad: Poƒçet minc√≠, poƒçet ud√°lost√≠, poƒçet √∫spƒõch≈Ø v s√©rii pokus≈Ø

## Binomick√© rozdƒõlen√≠
- Urƒçuje pravdƒõpodobnost ≈æe nastane urƒçit√Ω poƒçet √∫spƒõch≈Ø v urƒçit√©m poƒçtu pokus≈Ø, kde ka≈æd√Ω pokus m√° stejnou pravdƒõpodobnost √∫spƒõchu
- Rozdƒõlen√≠ si m≈Ø≈æeme zapamatovat dle zaƒç√°tku n√°zvu "bi", kter√© naznaƒçuje, ≈æe mohou nastat pouze dva p≈ô√≠pady (√∫spƒõch/ne√∫spƒõch)  

### Parametry
`k` = Poƒçet √∫spƒõch≈Ø, kter√Ω n√°s zaj√≠m√°  
`n` = Celkov√Ω poƒçet pokus≈Ø  
`p` = Pravdƒõpodobnost √∫spƒõchu v jednom pokusu (0 ‚â§ p ‚â§ 1)  

### Funkce v R
`dbinom(k, n, p)` = Vrac√≠ pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° s binomick√Ωm rozdƒõlen√≠m nabude p≈ôesnƒõ k √∫spƒõch≈Ø p≈ôi n pokusech, kde ka≈æd√Ω pokus m√° pravdƒõpodobnost √∫spƒõchu p  
`pbinom(k, n, p)` = Vrac√≠ pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° s binomick√Ωm rozdƒõlen√≠m nabude maxim√°lnƒõ k √∫spƒõch≈Ø (‚â§ k) p≈ôi n pokusech, kde ka≈æd√Ω pokus m√° pravdƒõpodobnost √∫spƒõchu p  

### P≈ô√≠klad (`dbinom`)
Jak√° je ≈°ance, ≈æe p≈ôi 10 hodech kostkou padne ≈°estka t≈ôikr√°t?
```r
k <- 3    # Poƒçet √∫spƒõch≈Ø (t≈ôikr√°t ≈°estka)
n <- 10   # Celkov√Ω poƒçet pokus≈Ø (hod≈Ø)
p <- 1/6  # Pravdƒõpodobnost ≈°estky v jednom pokusu (hodu)

pravdepodobnost <- dbinom(k, n, p) # Pravdƒõpodobnostn√≠ funkce
print(pravdepodobnost) # 0.1550454
```
Pravdƒõpodobnost, ≈æe p≈ôi 10 hodech kostkou padne ≈°estka p≈ôesnƒõ t≈ôikr√°t, je p≈ôibli≈ænƒõ 15,5 %.

### P≈ô√≠klad (`pbinom`)
Jak√° je pravdƒõpodobnost, ≈æe p≈ôi 10 hodech kostkou padne ≈°estka nejv√Ω≈°e t≈ôikr√°t?

```r
k <- 3    # Nejv√Ω≈°e t≈ôikr√°t ≈°estka
n <- 10   # Celkov√Ω poƒçet pokus≈Ø (hod≈Ø)
p <- 1/6  # Pravdƒõpodobnost ≈°estky v jednom pokusu (hodu)

pravdepodobnost <- pbinom(k, n, p) # Distribuƒçn√≠ funkce
print(pravdepodobnost) # 0.9302722
```
Pravdƒõpodobnost, ≈æe p≈ôi 10 hodech kostkou padne ≈°estka nejv√Ω≈°e t≈ôikr√°t, je p≈ôibli≈ænƒõ 93,0 %

## **Poissonovo rozdƒõlen√≠**  
- Urƒçuje pravdƒõpodobnost, ≈æe nastane k ud√°lost√≠ v pevnƒõ stanoven√©m ƒçasov√©m √∫seku nebo prostoru, pokud je pr≈Ømƒõrn√° frekvence v√Ωskytu zn√°m√°.

### Parametry 
- `k` = Poƒçet ud√°lost√≠, kter√Ω n√°s zaj√≠m√°
- `Œª (lambda)` = Pr≈Ømƒõrn√Ω poƒçet ud√°lost√≠ za jednotku ƒçasu nebo prostoru (Œª > 0)

### Funkce v R:
- `dpois(k, lambda)` = Vrac√≠ pravdƒõpodobnost, ≈æe nastane p≈ôesnƒõ k ud√°lost√≠ p≈ôi pr≈Ømƒõrn√© frekvenci v√Ωskytu **Œª**
- `ppois(k, lambda)` = Vrac√≠ pravdƒõpodobnost, ≈æe nastane nejv√Ω≈°e k ud√°lost√≠ p≈ôi pr≈Ømƒõrn√© frekvenci v√Ωskytu **Œª**


## P≈ô√≠klad (`dpois`)  
Jak√° je pravdƒõpodobnost, ≈æe urƒçit√Ωm bodem na silnici projede za hodinu p≈ôesnƒõ 20 aut, pokud pr≈Ømƒõrnƒõ projede 15 aut?

```r
k <- 20       # Poƒçet aut (p≈ôesnƒõ 20)
lambda <- 15  # Pr≈Ømƒõrn√Ω poƒçet aut za hodinu

pravdepodobnost <- dpois(k, lambda) # Pravdƒõpodobnostn√≠ funkce
print(pravdepodobnost) # 0.0516
```
Pravdƒõpodobnost, ≈æe urƒçit√Ωm bodem projede p≈ôesnƒõ 20 aut za hodinu, je p≈ôibli≈ænƒõ 5,2 %.

## P≈ô√≠klad (`ppois`)  
Jak√° je pravdƒõpodobnost, ≈æe urƒçit√Ωm bodem na silnici projede za hodinu nejv√Ω≈°e 20 aut, pokud pr≈Ømƒõrnƒõ projede 15 aut?

```r
k <- 20       # Nejv√Ω≈°e 20 aut
lambda <- 15  # Pr≈Ømƒõrn√Ω poƒçet aut za hodinu

pravdepodobnost <- ppois(k, lambda) # Distribuƒçn√≠ funkce
print(pravdepodobnost) # 0.8686
```
Pravdƒõpodobnost, ≈æe urƒçit√Ωm bodem projede nejv√Ω≈°e 20 aut za hodinu, je p≈ôibli≈ænƒõ 86,9 %.

## Hypergeometrick√© rozdƒõlen√≠
- Urƒçuje pravdƒõpodobnost, ≈æe p≈ôi **v√Ωbƒõru n prvk≈Ø bez vracen√≠** z celkov√©ho souboru bude pr√°vƒõ **k prvk≈Ø po≈æadovan√©ho typu**.

### Parametry
- `k` = Poƒçet √∫spƒõch≈Ø, kter√Ω n√°s zaj√≠m√° ve v√Ωbƒõru.  
- `w` = Poƒçet prvk≈Ø v kolekci, jejich≈æ v√Ωbƒõr pova≈æujeme za √∫spƒõch.  
- `b` = Poƒçet prvk≈Ø v kolekci, jejich≈æ v√Ωbƒõr pova≈æujeme za ne√∫spƒõch.  
- `n` = Poƒçet vybran√Ωch prvk≈Ø.

### Funkce v R

- `dhyper(k, w, b, n)` = Vrac√≠ pravdƒõpodobnost, ≈æe p≈ôi **v√Ωbƒõru n prvk≈Ø** bude p≈ôesnƒõ **k √∫spƒõch≈Ø** z celkov√©ho souboru s **w √∫spƒõ≈°n√Ωmi a b ne√∫spƒõ≈°n√Ωmi prvky**.  
- `phyper(k, w, b, n)` = Vrac√≠ pravdƒõpodobnost, ≈æe p≈ôi **v√Ωbƒõru n prvk≈Ø** bude **nejv√Ω≈°e k √∫spƒõch≈Ø** z celkov√©ho souboru s **w √∫spƒõ≈°n√Ωmi a b ne√∫spƒõ≈°n√Ωmi prvky**.  

## P≈ô√≠klad (`dhyper`)

## Geometrick√© rozdƒõlen√≠
- Urƒçuje pravdƒõpodobnost, ≈æe **prvn√≠ √∫spƒõch nastane po k ne√∫spƒõ≈°n√Ωch pokusech** v s√©rii nez√°visl√Ωch pokus≈Ø, kde ka≈æd√Ω pokus m√° stejnou pravdƒõpodobnost √∫spƒõchu **p**.  

### Parametry  
- `k` = Poƒçet ne√∫spƒõ≈°n√Ωch pokus≈Ø p≈ôed √∫spƒõ≈°n√Ωm pokusem (k ‚â• 0)  
- `p` = Pravdƒõpodobnost √∫spƒõchu v jednom pokusu (0 ‚â§ p ‚â§ 1) 

### Funkce v R  
- `dgeom(k, p)` = Vrac√≠ pravdƒõpodobnost, ≈æe prvn√≠ √∫spƒõch nastane `p≈ôesnƒõ po k ne√∫spƒõ≈°n√Ωch pokusech`.  
- `pgeom(k, p)` = Vrac√≠ pravdƒõpodobnost, ≈æe prvn√≠ √∫spƒõch nastane `nejv√Ω≈°e po k ne√∫spƒõ≈°n√Ωch pokusech`.  

## P≈ô√≠klad (`dgeom`)

```r

p <- 0.50    # Pravdƒõpodobnost √∫spƒõchu (hlava) v jednom hodu

pravdepodobnost <- dgeom(k, p) # Pravdƒõpodobnostn√≠ funkce
print(pravdepodobnost) # 0.0078
```
Pravdƒõpodobnost, ≈æe hlava padne poprv√© p≈ôi 8. pokusu, je p≈ôibli≈ænƒõ 0,78 %.

## P≈ô√≠klad (`pgeom`)

Pravdƒõpodobnost, ≈æe hlava padne nejv√Ω≈°e p≈ôi 8. pokusu, je p≈ôibli≈ænƒõ 99,2 %.

## Negativn√≠ (Pascalovo) binomick√© rozdƒõlen√≠
- Urƒçuje pravdƒõpodobnost, ≈æe **nastane k ne√∫spƒõch≈Ø p≈ôed dosa≈æen√≠m n √∫spƒõch≈Ø** v s√©rii nez√°visl√Ωch pokus≈Ø, kde ka≈æd√Ω pokus m√° pravdƒõpodobnost √∫spƒõchu **p**.  
- Je zobecnƒõn√≠m geometrick√©ho rozdƒõlen√≠ pro **v√≠ce ne≈æ jeden √∫spƒõch**.  

### Parametry  
- `k` = Poƒçet ne√∫spƒõch≈Ø, kter√© n√°s zaj√≠maj√≠ p≈ôed dosa≈æen√≠m `n` √∫spƒõch≈Ø (k ‚â• 0)
- `n` = Poƒçet po≈æadovan√Ωch √∫spƒõch≈Ø (n > 0)
- `p` = Pravdƒõpodobnost √∫spƒõchu v jednom pokusu (0 ‚â§ p ‚â§ 1) 

### Funkce v R  
- `dnbinom(k, n, p)` = Vrac√≠ pravdƒõpodobnost, ≈æe p≈ôed dosa≈æen√≠m **n √∫spƒõch≈Ø** nastane **p≈ôesnƒõ k ne√∫spƒõch≈Ø**.  
- `pnbinom(k, n, p)` = Vrac√≠ pravdƒõpodobnost, ≈æe p≈ôed dosa≈æen√≠m **n √∫spƒõch≈Ø** nastane **nejv√Ω≈°e k ne√∫spƒõch≈Ø**.  
- `1 - pnbinom(k, n, p)` = Vrac√≠ pravdƒõpodobnost, ≈æe p≈ôed dosa≈æen√≠m **n √∫spƒõch≈Ø** nastane **v√≠ce ne≈æ k ne√∫spƒõch≈Ø**.  

---

# Spojit√° rozdƒõlen√≠
- Popisuj√≠ spojit√© (nep≈ôetr≈æit√©) hodnoty
- Mezi dvƒõma libovoln√Ωmi hodnotami existuje nekoneƒçnƒõ mnoho dal≈°√≠ch hodnot
- Hodnoty mohou b√Ωt re√°ln√° ƒç√≠sla
- **Pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° nabude p≈ôesnƒõ jedn√© hodnoty, je 0**
- M√≠sto toho se pravdƒõpodobnost vyjad≈ôuje jako plocha pod k≈ôivkou hustoty pravdƒõpodobnosti v urƒçit√©m intervalu
- Spojit√° rozdƒõlen√≠ vy≈æaduj√≠ jasnƒõ definovan√Ω typ rozdƒõlen√≠ (nap≈ô. norm√°ln√≠, exponenci√°ln√≠) a v≈°echny pot≈ôebn√© parametry (st≈ôedn√≠ hodnota, rozptyl)

## Norm√°ln√≠ rozdƒõlen√≠` 
- Popisuje rozlo≈æen√≠ hodnot n√°hodn√© promƒõnn√© kolem st≈ôedn√≠ hodnoty (mean) se specifick√Ωm rozptylem (variance).

### Parametry  
- `Œº` = St≈ôedn√≠ hodnota (nap≈ô. pr≈Ømƒõrn√° v√Ω≈°ka)  
- `œÉ¬≤` = Rozptyl (variance)
- `œÉ` = Smƒõrodatn√° odchylka (standard deviation)

### Funkce v R
- `pnorm(x, mean, sd)` = Vrac√≠ pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° bude **men≈°√≠ nebo rovna x**
- `1 - pnorm(x, mean, sd)` = Vrac√≠ pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° bude **vƒõt≈°√≠ ne≈æ x**

---

## P≈ô√≠klad (`pnorm`)
Jak√° je pravdƒõpodobnost, ≈æe n√°hodnƒõ vybran√Ω mu≈æ je men≈°√≠ ne≈æ 170 cm, pokud je pr≈Ømƒõrn√° v√Ω≈°ka mu≈æ≈Ø 180 cm a rozptyl je 49 cm¬≤?

```r
mean <- 180    # St≈ôedn√≠ hodnota
sd <- sqrt(49) # Smƒõrodatn√° odchylka (odmocnina z rozptylu)
x <- 170       # Hledan√° v√Ω≈°ka

pravdepodobnost <- pnorm(x, mean, sd) # Distribuƒçn√≠ funkce
print(pravdepodobnost) # 0.1587
```
Pravdƒõpodobnost, ≈æe n√°hodnƒõ vybran√Ω mu≈æ je men≈°√≠ ne≈æ 170 cm, je p≈ôibli≈ænƒõ 15,9 %.

## P≈ô√≠klad (`1 - pnorm`)

## Lognorm√°ln√≠ rozdƒõlen√≠ 
- Popisuje hodnoty, jejich≈æ **logaritmus je norm√°lnƒõ rozdƒõlen**.  
- Hod√≠ se pro modelov√°n√≠ dat, kter√° jsou **kladn√° a maj√≠ v√Ωraznƒõ pravostrann√© rozlo≈æen√≠** (nap≈ô. p≈ô√≠jmy, ceny akci√≠). 

### Parametry  
- `Œº` = St≈ôedn√≠ hodnota logaritmovan√© promƒõnn√©  
- `œÉ` = Smƒõrodatn√° odchylka logaritmovan√© promƒõnn√©  

### Funkce v R
- `dlnorm(x, meanlog, sdlog)` = Vrac√≠ pravdƒõpodobnost hustoty pro hodnotu `x` s dan√Ωmi parametry  
- `plnorm(x, meanlog, sdlog)` = Vrac√≠ pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° bude **men≈°√≠ nebo rovna x**  
- `qlnorm(p, meanlog, sdlog)` = Vrac√≠ hodnotu odpov√≠daj√≠c√≠ pravdƒõpodobnosti `p` 
- `rlnorm(n, meanlog, sdlog)` = Generuje `n` n√°hodn√Ωch hodnot z lognorm√°ln√≠ho rozdƒõlen√≠  

### P≈ô√≠klad (`plnorm`)

## Exponenci√°ln√≠ rozdƒõlen√≠  
- Popisuje **ƒças nebo vzd√°lenost mezi dvƒõma po sobƒõ jdouc√≠mi ud√°lostmi**, kter√© nast√°vaj√≠ **nez√°visle a konstantn√≠ pr≈Ømƒõrnou rychlost√≠**. 

### Parametry
- `Œª (lambda)` = Intenzita ud√°losti *(poƒçet ud√°lost√≠ za jednotku ƒçasu nebo prostoru, Œª > 0)*  

### Funkce v R  
- `dexp(x, rate)` = Vrac√≠ **hustotu pravdƒõpodobnosti** pro hodnotu `x` s parametrem `Œª`
- `pexp(x, rate)` = Vrac√≠ pravdƒõpodobnost, ≈æe n√°hodn√° promƒõnn√° bude **men≈°√≠ nebo rovna x** 
- `qexp(p, rate)` = Vrac√≠ hodnotu, kter√° odpov√≠d√° pravdƒõpodobnosti `p` *(kvantil)*
- `rexp(n, rate)` = Generuje `n` n√°hodn√Ωch hodnot z exponenci√°ln√≠ho rozdƒõlen√≠

## P≈ô√≠klad (`pexp`)

